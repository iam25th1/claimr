Flow rescue batch
==================

What this fixes (from the live audit, in priority order)

  1. Content centering. Project + dashboard layouts now wrap children in
     max-w-6xl mx-auto. Forms (settings) explicitly narrower at max-w-2xl
     inside the wrapper.

  2. Double-sidebar bug. /project/post, /project/jobs, /project/escrow,
     and /project/analytics each rendered ProjectSidebar + their own
     <main className="ml-64"> wrapper on TOP of the layout's. Content was
     pushed 128px off-center. Each sub-page now returns just its content.

  3. WalletAddressCard shows live USDC balance. Visible everywhere the
     card lives: /project, /dashboard/wallet, /project/escrow. Funded
     users finally see the result.

  4. Sidebars stop lying. "Connected" replaced with "Sign in" when no
     auth. Both creator-side and project-side fixed.

  5. /project/settings rebuilt. Real form: name, tagline, website, X
     handle, description. Saves to localStorage keyed by wallet. The
     ArcSwap/team@arcswap hardcoded mock is gone.

  6. /dashboard/settings rebuilt. Real form: display name, X handle,
     bio, payout address, notification toggles. The jenzy.eth / Turnkey
     mock is gone. (Turnkey reference was wrong - wallet is Circle.)

  7. Navbar "Launch App" routes based on auth state. Signed in goes to
     /dashboard/discover. Signed out gets sent to /onboarding?role=creator
     and the button label reads "Sign in" instead.

  8. Onboarding has a "we create a wallet for you, no MetaMask needed"
     callout below the subheading.

  9. Em dash scrub on user-facing strings (my-jobs-list pending text).
