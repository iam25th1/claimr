MetaMask UX hardening

What was happening

  "Provider not found. Version: @wagmi/core@3.4.12"
  wagmi can't connect because there's no window.ethereum in your browser.
  Either MetaMask is not installed, is locked, or you're in a browser
  without an extension. The connect button fired anyway, hit the error,
  and surfaced wagmi's terse internal message to the user.

  "Error with Permissions-Policy header: Unrecognized feature: interest-cohort"
  Benign. Some server header still references the deprecated FLoC
  opt-out. Chrome dropped the feature name; the warning is cosmetic.
  Doesn't affect anything in your app. Leave it alone or, if it really
  bothers you, find where the header is set (vercel.json or
  next.config.js headers()) and remove "interest-cohort=()" from the
  Permissions-Policy value.


This batch

  components/claimr/onboarding-cards.tsx           PATCHED

  Two surgical changes:

  1. Wallet detection on mount.
     EmailStep now runs a useEffect on mount that checks
     `window.ethereum`. The result is tri-state (null while detecting,
     true if present, false if absent) and drives the button render:

       null    button shown but disabled while detecting (~10ms)
       true    "Connect MetaMask" button enabled
       false   "Install MetaMask" link to metamask.io download page

     No one clicks a button that's going to fail. Users without a
     wallet see exactly what they need: an install link.

  2. Translated error messages.
     handleConnectWallet now catches the connect error and translates
     wagmi's internal phrasing into something a user can act on:

       "provider not found"          -> "No browser wallet detected.
                                         Install MetaMask or another
                                         browser wallet and refresh."
       "user rejected"               -> "Wallet connection was cancelled."
       "already pending"             -> "A wallet connection request is
                                         already open. Check your
                                         MetaMask extension."
       anything else                 -> raw message (truncated wagmi
                                         output is still better than
                                         nothing)
