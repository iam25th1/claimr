Mobile polish

Quick note: the main mobile pass is already in your repo. Both layouts
use a shared MobileTabBar (bottom tabs + overflow More sheet with sign
out), sidebars are hidden md:flex, safe-area inset handled, animated
sheet via motion. Good work already shipped.

What this small zip fixes are the two surfaces that still felt cramped
on phones after that work:

  components/claimr/wallet-address-card.tsx
    The 3 buttons (Copy, Arcscan, Fund wallet) now stack vertically
    full-width on phones, side-by-side on sm+. Each button gets its own
    row on mobile so they're tap-sized.

  components/claimr/page-header.tsx
    Title block + action button were using flex-wrap which made the
    action button drift awkwardly on narrow widths. Now properly stacks
    vertically on mobile (title -> action below, full-width), side-by-
    side on sm+. The compound selectors on the action wrapper force any
    Link or button child to fill the row and center its content.
